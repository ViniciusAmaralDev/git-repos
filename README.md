# Git Repos - Aplicação em react native que consome a api do github e lista os repositórios, podendo interagir e muito mais.

#### Escolha uma das opções abaixo e faça o clone o projeto:
> HTTPS
```BASH
git clone https://github.com/ViniciusAmaralDev/git-repos.git
```
> SSH
```BASH
git clone git@github.com:ViniciusAmaralDev/git-repos.git
```

#### Após fazer o clone do projeto, instale as dependências:
> YARN
```BASH
yarn install
```
> NPM
```BASH
npm install
```

> Se você for rodar em um emulador iOS, execute:
```BASH
npx pod-install
ou
cd ios && pod install && cd ..
```

#### Agora você está pronto para rodar o projeto, execute:
```BASH
npx expo start
```
> Ou

iOS:
```BASH
yarn android
ou
yarn ios
```
Android:
```BASH
npm run android
ou
npm run ios
```
